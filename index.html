Как пользоваться

1. Введите нужное смещение воздуховода (расстояние, на которое нужно обойти препятствие).

2. Выберите угол полуотводов (чаще всего 45°).

3. Укажите диаметр труб и длину полуотвода.

4. Приложение покажет точную длину патрубка для реза.

Формула расчёта

Осевая длина: L = S × K

Где S — смещение, K — коэффициент угла.

Длина реза: L_cut = L − 2×вылет + 2×нахлёст

Как определить вылет полуотвода

Приложите два полуотвода друг к другу «уткой» и замерьте общее смещение без патрубка. Разделите на 2 — это вылет одного полуотвода.

Если не знаете вылет — оставьте поле пустым. Расчёт покажет осевую длину без учёта вылетов.

Нахлёст (ниппель)

Для ниппельного соединения стандартный заход — 30–50 мм с каждой стороны. Для зигованного — зависит от глубины зига (обычно 15–25 мм).

Пример

Смещение 200 мм, угол 45°, вылет полуотвода 120 мм, нахлёст 50 мм.

L = 200 × 1.41 = 282 мм

L_cut = 282 − 2×120 + 2×50 = 262 мм
